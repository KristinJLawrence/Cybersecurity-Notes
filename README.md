# Cybersecurity Notes

Personal cybersecurity notes covering networking, Linux, security tools, and web vulnerabilities.

## Topics Covered

* Networking fundamentals
* Linux basics
* Security tools
* Web security vulnerabilities
* Cryptography
* CTF writeups

## Purpose

This repository documents my learning journey in cybersecurity. The notes are written while studying different security concepts, tools, and techniques during my coursework and independent study.

## Structure

The notes are organized by topic to make them easy to navigate:

* `networking/` – Networking concepts, protocols, and troubleshooting
* `linux/` – Linux commands, system administration, and security
* `tools/` – Notes on security tools such as Nmap, Wireshark, and Burp Suite
* `web-security/` – Common web vulnerabilities and attack techniques
* `cryptography/` – Encryption, hashing, and security concepts
* `cheatsheets/` – Quick reference commands and summaries

## Disclaimer

These notes are for **educational purposes only** and are based on personal learning and coursework.

## Goals

* Reinforce cybersecurity concepts through documentation
* Build a structured knowledge base
* Share useful learning resources with others
