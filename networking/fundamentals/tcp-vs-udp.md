# TCP vs UDP

TCP (Transmission Control Protocol) and UDP (User Datagram Protocol) are transport layer protocols.

## TCP

- Connection-oriented
- Reliable delivery
- Error checking and retransmission
- Slower but dependable

Common TCP protocols:
- HTTP / HTTPS
- FTP
- SMTP

## UDP

- Connectionless
- Faster communication
- No guarantee of delivery
- Used where speed is more important than reliability

Common UDP protocols:
- DNS
- VoIP
- Streaming
- Online gaming
